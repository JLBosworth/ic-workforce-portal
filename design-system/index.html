02
Typography
Two typefaces. One role each. Never crossed.
DM Serif Display — Headlines, card titles, statistics, hero text, and any number that deserves weight. Loaded via Google Fonts at italic and normal. Never used for UI chrome, labels, or body copy.

Plus Jakarta Sans — All UI text: navigation, buttons, labels, body, meta, tags, inputs. Weights 300–700 in use. Never used for display headlines.

48px base grid throughout

Rules & Constraints
✓ Do
Use CSS custom properties for all colors
DM Serif Display for all headlines and stats
Plus Jakarta Sans for all UI and body text
Sharp corners (0px) for most components
48px horizontal page padding
Terra as the primary CTA color
Slate as the nav, footer, hero, and banner background
Uppercase + letter-spacing for all labels and eyebrows
Italic DM Serif for accent phrases in headlines
Left-border accents for card tier/status color coding
✕ Don't
Hardcode hex values in HTML or JS
Use inline style="" attributes in markup
Use DM Serif for body, buttons, or labels
Use Plus Jakarta Sans for display headlines
Use rounded corners larger than 6px
Add orange/terra gradients to the hero
Use border-radius on filter chips or cards
Use DM Serif at weights other than 400 (italic or upright)
Add drop shadows larger than 0 4px 20px
Introduce new typefaces or colors outside the token set
